# Contributing

Contributions are welcome. Here's how to get started.

## Setup

```bash
git clone https://github.com/JDRV-space/mech-sound.git
cd mech-sound
open MechSound.xcodeproj
```

## Development

- Swift source follows Apple's Swift style guidelines
- Build and run on macOS 12+
- Test with multiple keyboard layouts before submitting

## Pull Requests

1. Fork the repo
2. Create a feature branch (`git checkout -b my-feature`)
3. Make your changes
4. Test on your Mac
5. Commit with a clear message
6. Open a PR

## Reporting Bugs

Open an issue with:
- macOS version (`sw_vers`)
- Mac model (`sysctl hw.model`)
- Steps to reproduce
